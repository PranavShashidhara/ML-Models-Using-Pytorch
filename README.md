## How to Run
1. Clone this repo:
```bash
  > git clone git@github.com:PranavShashidhara/ML_ops_deployment.git
   cd ML-Models-Using-Pytorch
```
2. Install dependencies: 
```bash 
> pip install torch numpy matplotlib scikit-learn
```
3. Open Jupyter Lab/Notebook: 
```bash 
> jupyter lab
```
